# Barcode-Scanner-Emulator

A Simple Barcode Scanner Device "Emulator" Built using python and MIT App Inventor

  - Act like a generic barcode scanner device
  - Use your mobile phone as the scanner
  - Magic

## Dependencies

For python client
```sh
python ^3.x 
pyautogui
```
